# Invoice Ninja Container

Alternative container images for [Invoice Ninja](https://invoiceninja.com/), a self-hosted invoicing, billing, and payment platform.

Sources are available on [GitHub](https://github.com/anthochamp/container-invoiceninja).

See [README.md](README.md) for full documentation.

## Image tags

- `x.y.z-invoiceninjaA.B.C`: Container image version `x.y.z` with Invoice Ninja `A.B.C`.
- `edge-invoiceninjaA.B.C`: Latest commit build with Invoice Ninja `A.B.C`.

**Tag aliases:**

- `x.y-invoiceninjaA.B.C`: Latest patch of `x.y` (major.minor) with Invoice Ninja `A.B.C`.
- `x-invoiceninjaA.B.C`: Latest minor+patch of `x` (major) with Invoice Ninja `A.B.C`.
- `x.y.z`: Version `x.y.z` with latest Invoice Ninja (only latest container version updated).
- `x.y`: Latest patch of `x.y` (major.minor) with latest Invoice Ninja (only latest container major.minor updated).
- `x`: Latest minor+patch of `x` (major) with latest Invoice Ninja (only latest container major updated).
- `invoiceninjaA.B`: Latest container with latest patch of Invoice Ninja `A.B` (major.minor).
- `invoiceninjaA`: Latest container with latest minor+patch of Invoice Ninja `A` (major).
- `latest`: Latest `x.y.z-invoiceninjaA.B.C` tag.
- `edge-invoiceninjaA.B`: Latest commit build with latest patch of Invoice Ninja `A.B` (major.minor).
- `edge-invoiceninjaA`: Latest commit build with latest minor+patch of Invoice Ninja `A` (major).
- `edge`: Latest `edge-invoiceninjaA.B.C` tag.
