# Invoice Ninja Container

Alternative container images for [Invoice Ninja](https://invoiceninja.com/), a self-hosted invoicing, billing, and payment platform.

Sources are available on [GitHub](https://github.com/anthochamp/container-invoiceninja).

See [README.md](README.md) for full documentation.

## Image tags

- `x.y.z-invoiceninjaA.B.C`: Container image version `x.y.z` with Invoice Ninja `A.B.C`.
- `edge-invoiceninjaA.B.C`: Latest commit build with Invoice Ninja `A.B.C`.

**Tag aliases:**

- `x.y-invoiceninjaA.B.C`: Latest patch of `x.y` with Invoice Ninja `A.B.C`.
- `x-invoiceninjaA.B.C`: Latest minor+patch of `x` with Invoice Ninja `A.B.C`.
- `x.y.z-invoiceninjaA.B`: Version `x.y.z` with latest patch of Invoice Ninja `A.B` (only latest container version updated).
- `x.y-invoiceninjaA.B`: Latest patch of `x.y` with latest patch of Invoice Ninja `A.B`.
- `x-invoiceninjaA.B`: Latest minor+patch of `x` with latest patch of Invoice Ninja `A.B`.
- `x.y.z-invoiceninjaA`: Version `x.y.z` with latest minor+patch of Invoice Ninja `A` (only latest container version updated).
- `x.y-invoiceninjaA`: Latest patch of `x.y` with latest minor+patch of Invoice Ninja `A`.
- `x-invoiceninjaA`: Latest minor+patch of `x` with latest minor+patch of Invoice Ninja `A`.
- `x.y.z`: Version `x.y.z` with latest Invoice Ninja (only latest container version updated).
- `x.y`: Latest patch of `x.y` with latest Invoice Ninja.
- `x`: Latest minor+patch of `x` with latest Invoice Ninja.
- `invoiceninjaA.B.C`: Latest container with Invoice Ninja `A.B.C`.
- `invoiceninjaA.B`: Latest container with latest patch of Invoice Ninja `A.B`.
- `invoiceninjaA`: Latest container with latest minor+patch of Invoice Ninja `A`.
- `latest`: Latest `x.y.z-invoiceninjaA.B.C` tag.
- `edge-invoiceninjaA.B`: Latest commit build with latest patch of Invoice Ninja `A.B`.
- `edge-invoiceninjaA`: Latest commit build with latest minor+patch of Invoice Ninja `A`.
- `edge`: Latest `edge-invoiceninjaA.B.C` tag.
